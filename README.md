## What is EmuRank?

EmuRank is a web platform where users submit and vote on emulation compatibility reports for video games. It automatically determines the best-known compatibility level for each game based on the most relevant and up-to-date community data.

- Browse compatibility reports for games across 14 platforms and 50+ emulators
- Submit your own reports with emulator version, OS, hardware specs, and compatibility level
- Vote on existing reports to signal their accuracy
- View emulator statistics and compatibility trends
